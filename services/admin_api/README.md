# Admin CMS API

This backend API is created for SWAPI-SWAP Admin CMS panel.

## API end points

### Admin API endpoints

All the API endpoints will be followed by `${rootUrl}/api/v1/admin`

| #   | API | Method | Description       |
| --- | --- | ------ | ----------------- |
| 1.  | `/` | GET    | Get all admins    |
| 2.  | `/` | POST   | Create new admins |
